Thanks for installing {{ .Chart.Name }} {{ .Chart.AppVersion }}.

Finish setup in four steps:

1. Retrieve the temporary admin password generated during install:

     kubectl get secret {{ include "foggy.adminSecretName" . }} -n {{ .Release.Namespace }} \
       -o jsonpath='{.data.password}' | base64 -d ; echo

{{- if .Values.ingress.enabled }}

2. Access the Foggy Console at:

     http{{ if .Values.ingress.tls.enabled }}s{{ end }}://{{ .Values.ingress.host }}
{{- else }}

2. Port-forward the Console Service to your local machine:

     kubectl port-forward -n {{ .Release.Namespace }} svc/{{ .Release.Name }}-console 8080:{{ .Values.consoleFrontend.service.port }}

   Then open http://localhost:8080 in your browser.
{{- end }}

3. Log in with email `admin@foggy.local` and the temporary password from step 1.
   You will be redirected through a short setup sequence; each step
   auto-advances to the next:
     - first-login: set your own admin email and a permanent password (the
       bootstrap admin is replaced; subsequent logins use your chosen email)
     - license-activate: paste the Foggy license JWT (self-hosted installs
       require an active license for mutating API calls; read-only endpoints
       remain available without one)
     - llm-provider-setup: paste your Anthropic API key (OpenAI, Azure OpenAI,
       and OpenRouter are also supported; configurable post-install)

4. After setup you land on your workspace dashboard. Click "Connectors" in
   the left sidebar to wire up data sources. Kubernetes is already connected
   via the Agent pod's in-cluster ServiceAccount; add Grafana, Slack, GitHub,
   Linear, Jira, Sentry, Prometheus, Loki, or Webhook as needed.

Docs:  https://docs.foggyhq.com/self-hosted/getting-started
RBAC:  https://docs.foggyhq.com/self-hosted/rbac
Help:  support@foggyhq.com
